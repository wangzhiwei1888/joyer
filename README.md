# joyer 前端构建工具

### 安装

`npm install -g joyer`

`joyer -V`

`joyer -h`

注：国内建议用淘宝镜像：`npm --registry=https://registry.npm.taobao.org ` 或者用阿里提供的[cnpm](https://npm.taobao.org/)

### 创建工程
 
 `joyer init myapp`
 
 
### 开发

 `cd myapp`
 
 `joyer dev`
 
 启动服务在[http://localhost:3000/demo/index.html](http://localhost:3000/demo/index.html)
 
### 编译

`joyer build`

### 发布

`joyer deploy`
